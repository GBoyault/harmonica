# SUPER BLUES HARP SIMULATOR 2022
A basic harmonica simulator, using FM synthesis with javascript Web Audio API to generate the sound.

## Installation

In the project directory, run:

### `npm install`

## Available Scripts

In the project directory, you can run:

### `npm run watch`

Runs the app in the development mode.\
Open [http://localhost:3000/harmonica](http://localhost:3000/harmonica) to view it in your browser.

The page will reload when you make changes.\
You can adapt the location of your proxy project with your local env by updating `paths.proxy` in the `gulpfile.babel.js` file.

### `npm run build`

Builds the app for production to the `dist` folder.\

The build is minified and optimized.\
